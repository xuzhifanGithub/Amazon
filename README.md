## 安装与部署

### 1. 克隆仓库
  本项目包含较大的二进制文件（AI 引擎 `kataAmazon.exe`），使用 **Git LFS** 进行管理。
  1)安装 Git LFS 
  2)克隆仓库
  3)验证大文件是否下载成功

### 2. Python 环境要求
  执行mcts算法的python版本要求:3.11.5/3.13.3/自行编译
